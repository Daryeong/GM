# Game UI/UX Portfolio

최다령의 게임 UI/UX 포트폴리오 정적 웹사이트입니다.

## 구성

- `index.html`: 포트폴리오 콘텐츠
- `styles.css`: 레이아웃과 반응형 스타일
- `script.js`: 모바일 내비게이션
- `assets/original/`: PNG 및 PDF 원본 작업물

## 로컬 실행

별도의 빌드 과정 없이 `index.html`을 브라우저에서 열어 확인할 수 있습니다.

## 배포

`main` 브랜치에 푸시하면 GitHub Actions가 GitHub Pages로 자동 배포합니다.
