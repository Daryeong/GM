# Game UI/UX Portfolio

## 구조
- `index.html`: 포트폴리오 콘텐츠
- `styles.css`: 전체 스타일과 반응형 설정
- `script.js`: 모바일 메뉴
- `assets/original/`: 업로드한 기존 PNG/PDF 원본 포함

## 수정 포인트
- 이메일은 `index.html`의 `mailto:your-email@example.com`를 실제 주소로 교체하세요.
- 지원 정보에는 지원 분야, 작업 범위, 사용 툴, 연락처를 넣는 구성으로 정리했습니다.

## 변경 사항
- 전체 문구를 더 자연스러운 포트폴리오 말투로 수정했습니다.
- `지원 정보` 섹션을 지원 분야와 작업 범위가 보이도록 정리했습니다.
- 기존 이미지와 PDF 파일 경로는 유지했습니다.


## Design update
- 참고 PDF의 큰 타이포그래피, 여백, 카드형 정보 정리 방향만 반영했습니다.
- 기존 색감, 카드 형태, 이미지 리소스 표현 방식은 유지했습니다.

## 추가 변경 사항
- 지원 정보의 메일 버튼을 패션 포트폴리오 링크 2개로 교체했습니다.


- 상단 고정 메뉴에 홈으로 링크를 추가했습니다.
